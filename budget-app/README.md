# Budgeting
The application is for people to help them track their spendings. It is mostly for people who like to see where they use their money mostly and help them save money. The application is a great way to plan ahead and see how much are you earning and how much will you use for necessities.

A user can create an account and log to their interest their spendings. A user sees how much money have they spent.

#### How to install and get the app working

Firstly install poetry (poetry install), and then run the application with "poetry run invoke start"


#### Command line commands:

For running the app use: poetry run invoke start

For testing run: poetry run invoke test

For report for testing run: poetry run invoke coverage-report
